dvdauthor and spumux

Spumux is an opensource utility to mux subtitles or menu buttons into an mpeg2 stream. It comes with the dvdauthor package available at http://dvdauthor.sourceforge.net

I use a very useful plugin for gimp by Jamie Strachan to generate the images and config file for spumux. You can get it at the gimp plugin page or directly here.

Using dvdauthor to author dvd's can sometimes be frustrating because some things are not that apparent from the start. The following are a few hints and tips to explain dvdauthor and spumux in more detail. I hope it will be helpful.
